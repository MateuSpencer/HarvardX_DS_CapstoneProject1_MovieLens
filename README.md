# HarvardX Data Science Capstone — MovieLens Recommendation

**Course:** HarvardX Professional Certificate in Data Science
**Project:** Capstone Project 1

## Description

A **movie recommendation system** built on the MovieLens dataset. Implements and evaluates multiple approaches to predict user ratings, including user and movie effects, regularization, and matrix factorization.

The project includes a full report with exploratory data analysis, model building, evaluation (RMSE), and conclusions.

## Technologies

- **Language:** R
- **Libraries:** tidyverse, caret, data.table
- **Output:** PDF report (R Markdown)

## Project Structure

```
Movie_Lens_Script.R       # Main analysis and model script
Movie_Lens_Report.Rmd     # R Markdown report source
Movie_Lens_Report.pdf     # Final compiled report
```
